Loads the jQuery Backstretch plugin and provides an easy GUI for it.

== Description ==

wp-backstretch loads jQuery Backstretch into WordPress and provides an easy GUI to select an image to be stretched across your background.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates

== Frequently asked questions ==

= How do I set the background? =

1. Upload an image at /wp-admin/media-new.php and copy the URL.
1. Go to /wp-admin/themes.php?page=wp-backstretch and paste in the image URL.
1. Lean back and sip some coffee

= How do I use the shortcode? =
1. Upload an image at /wp-admin/media-new.php and copy the URL.
1. Go and edit the page where you want to set a custom background.
1. Write [backstretch url="http://theimageurlhere"] anywhere on that page
1. Save the page
